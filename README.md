## Страницы

| Маршрут | Страница |
|---|---|
| `/` | Вход |
| `/dashboard` | Главная панель со статистикой |
| `/clients` | Список клиентов |
| `/clients/[id]` | Карточка клиента |
| `/orders` | Список заказов + фильтры |
| `/orders/[id]` | Карточка заказа + статус-бар |
| `/finances` | Доходы / расходы / графики |
| `/services` | Услуги и прайс |

## Стек

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- JetBrains Mono (Google Fonts)
